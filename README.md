# Express Starter

> A Starter Repo to create a Rest API using Express

[![Commitizen friendly](https://img.shields.io/badge/commitizen-friendly-brightgreen.svg)](http://commitizen.github.io/cz-cli/)

## Development

- Clone the repo

```bash
$ git clone https://github.com/matalmeida/restapi-express.git
```

- Install dependencies

```bash
$ npm i
```

- Run scripts

| Action                 | Usage          |
| ---------------------- | -------------- |
| Start development mode | `npm run dev`  |
| Start production mode  | `npm start`    |
| Lint code              | `npm run lint` |

- Commiting your code

> This starter has a configured `git-commitizen`.
> After you stage your changes you can simply run `git commit` to follow the commit convention.

## Environment

```yaml
# .env file
PORT=3000
```

## Endpoints

| Method | Endpoint       | Description                   |
| ------ | -------------- | ----------------------------- |
| GET    | `/api/v1`      | Get avaliable routes          |
| GET    | `/api/v1/docs` | Get docs generated by Swagger |
